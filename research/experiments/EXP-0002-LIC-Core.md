# EXP-0002: LIC Core

## Тема

Проверка LIC Core как каркаса из нескольких точек.

## Постановка

Первоначальная идея LIC могла выглядеть как поиск одной стабильной длины. Однако первые эксперименты показывают, что устойчивость может принадлежать не одной величине, а группе точек и отношений.

Возможные точки LIC Core:

- `left_eye_inner`
- `right_eye_inner`
- `nose_bridge`
- `nose_tip`
- `chin`
- `mouth_center`

## Метрики

- `detection_rate`
- `position_std`
- `mean_displacement`
- `coefficient_of_variation` для расстояний
- сравнение `passed`, `warning`, `rejected`

## Следующий шаг

Проверить LIC Core на разных людях и разных видео, не ограничиваясь одной серией кадров.

## Ограничения

LIC Core пока является исследовательской гипотезой. Он не является инструментом оценки личности, идентификации или биометрической аутентификации.
