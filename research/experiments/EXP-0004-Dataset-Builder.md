# EXP-0004: Dataset Builder

## Тема

Dataset Builder как инструмент массового получения портретов.

## Зачем нужны видео

Видео позволяет получить серию кадров одного человека при изменении мимики, поворота головы, освещения и качества изображения. Это важно для проверки устойчивости landmark и расстояний.

## Источники

Кино, интервью и другие материалы с крупными планами лиц могут быть полезны как источник экспериментальных серий. При этом источник, параметры извлечения кадров и ограничения должны фиксироваться в описании эксперимента.

## Проблема дублей

Последовательные кадры видео часто почти одинаковы. Без дедупликации они могут искусственно завышать уверенность статистики.

## Фильтрация качества

Dataset Builder должен отделять `passed`, `warning`, `rejected` и `duplicates`, чтобы исследователь видел качество материала, а не только итоговое количество изображений.

## Вывод

Dataset Builder является научным прибором для Profile: он помогает получать серии наблюдений, необходимые для проверки LIC и устойчивости геометрии лица.
