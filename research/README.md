# Research

`research/` — исследовательский дневник проекта Profile. Этот раздел не заменяет `README.md` и не является технической документацией по запуску кода. Его задача — фиксировать развитие гипотез, методологии, экспериментов и терминологии вокруг геометрии лица, landmark, LIC и воспроизводимости измерений.

Profile пока является исследовательским проектом и не должен использоваться для оценки личности, профессиональной пригодности, интеллекта, характера или надежности человека. Проект также не предназначен для психологических, HR, криминалистических или биометрических выводов по лицу.

## Карта раздела

- [000_Manifest.md](000_Manifest.md) — манифест исследовательского направления.
- [001_Origin.md](001_Origin.md) — история появления идеи LIC и перехода к инвариантной геометрии лица.
- [002_Hypotheses.md](002_Hypotheses.md) — текущий список гипотез и их статусы.
- [003_Methodology.md](003_Methodology.md) — методология экспериментов и базовый цикл проверки.
- [experiments/](experiments/) — постановки первых экспериментов.
- [lic/](lic/) — документы по LIC, LIC Core, каноническому лицу и графовой модели.
- [dataset/](dataset/) — документы по Dataset Builder, правилам датасета и критериям качества.
- [glossary.md](glossary.md) — словарь терминов.
- [timeline.md](timeline.md) — хронология развития исследовательского направления.
- [changelog.md](changelog.md) — журнал изменений research-раздела.
