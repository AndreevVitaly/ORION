# Knowledge

`knowledge/` — граф знаний проекта Profile. Здесь фиксируются идеи, открытые вопросы, принятые исследовательские решения и будущие направления развития.

Этот раздел не является технической документацией и не заменяет эксперименты. Его задача — сохранять исследовательскую память проекта так, чтобы развитие LIC, Dataset Builder и методологии оставалось прослеживаемым.

Profile не делает выводов о личности, характере, интеллекте, профессиональной пригодности или надежности человека. Документы `knowledge/` описывают только исследовательские идеи и решения о геометрии лица, landmark, устойчивости и воспроизводимости.

## Реестры

- [ideas.md](ideas.md) — идеи проекта.
- [questions.md](questions.md) — открытые вопросы.
- [decisions.md](decisions.md) — принятые исследовательские решения.
- [future.md](future.md) — карта будущих направлений.

## Правило идентификаторов

Каждая значимая идея, вопрос или решение получает уникальный идентификатор.

Форматы:

- `IDEA-0001`
- `Q-0001`
- `DEC-0001`
