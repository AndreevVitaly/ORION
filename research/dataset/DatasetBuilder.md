# Dataset Builder

Dataset Builder предназначен для формирования исследовательских датасетов Profile. Он помогает получать серии портретов из локального видео, YouTube URL или набора изображений.

Базовый процесс:

```text
видео -> кадры -> лица -> crop -> portrait.json -> датасет
```

Dataset Builder должен быть отдельным приложением, потому что его задача отличается от математического ядра `portrait_core`. Он отвечает за извлечение материала, фильтрацию качества, дедупликацию и подготовку структуры эксперимента.

Dataset Builder не должен идентифицировать людей. Его задача — подготовка геометрических наблюдений для исследования устойчивости landmark, LIC и отношений лица.

Проект не использует Dataset Builder для психологических, HR, криминалистических или биометрических выводов по лицу.
