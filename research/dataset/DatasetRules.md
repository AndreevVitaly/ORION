# Dataset Rules

## Правила датасета

- Хранить исходник эксперимента или ссылку на него, если это допустимо.
- Фиксировать источник данных.
- Фиксировать шаг кадров и параметры извлечения.
- Разделять `passed`, `warning`, `rejected` и `duplicates`.
- Не смешивать людей без маркировки.
- Не делать выводы по одному человеку.
- Хранить `summary.json`.
- Не хранить лишние персональные данные.

## Ограничения

Датасет Profile предназначен для проверки геометрических гипотез. Он не должен использоваться для оценки личности, профессиональной пригодности, интеллекта, характера, надежности или идентификации человека.
