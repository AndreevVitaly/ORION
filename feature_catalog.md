# Каталог признаков v1

Все признаки рассчитываются в канонической системе
`eye-centered-interocular`: центр между глазами является началом координат,
межцентровое расстояние глаз равно `1`, roll устранен.

| Признак | Зона | Формула | Единица | Минимум вершин |
|---|---|---|---|---:|
| `face_bbox_ratio` | лицо | ширина bounding box / высота | отношение | 3 |
| `face_hull_area` | лицо | площадь выпуклой оболочки | каноническая площадь | 3 |
| `face_hull_perimeter` | лицо | периметр выпуклой оболочки | каноническая длина | 3 |
| `face_compactness` | лицо | 4π × площадь / периметр² | оценка 0–1 | 3 |
| `face_mirror_error` | лицо | среднее расстояние до ближайшей зеркальной вершины | каноническая длина | 3 |
| `lip_outer_height` | внешний контур губ | полная высота внешнего контура | каноническая длина | 3 |
| `mouth_opening_height` | ротовая щель | высота внутреннего контура | каноническая длина | 3 |
| `mouth_opening_area` | ротовая щель | площадь внутреннего контура | каноническая площадь | 3 |
| `lip_tissue_area` | губы | внешняя площадь минус площадь ротовой щели | каноническая площадь | 3 |
| `upper_lip_curvature` | верхняя губа | длина дуги / хорда − 1 | оценка | 3 |
| `lower_lip_curvature` | нижняя губа | длина дуги / хорда − 1 | оценка | 3 |
| `lip_mirror_error` | губы | ошибка зеркальной симметрии контура | каноническая длина | 3 |
| `left_eye_density` | левый глаз | число вершин в зоне* | количество | 1 |
| `right_eye_density` | правый глаз | число вершин в зоне* | количество | 1 |
| `nose_bbox_ratio` | нос | ширина bounding box / высота | отношение | 3 |
| `mouth_bbox_ratio` | рот | ширина bounding box / высота | отношение | 3 |
| `jaw_hull_area` | челюсть | площадь выпуклой оболочки | каноническая площадь | 3 |

## Уверенность

Уверенность признака раскрывается на компоненты:

- `landmark_validity` — координаты конечны и пригодны для вычислений;
- `pose` — допустимость ракурса;
- `zone_coverage` — полнота нужного контура или зоны;
- `image_quality` — проверки исходного кадра;
- `stability` — повторяемость по серии, до серии имеет значение `null`.

Yaw и pitch пока являются диагностическими 2D-прокси, а не метрическими
углами. Они снижают доверие, но не используются для искусственного
восстановления формы лица.

`*` Density-признаки служат диагностикой покрытия сетки и сравниваются только
внутри одной топологии. Они исключены из межсерийной морфологической
стабильности.

## Роли

- `morphology` — постоянная геометрия; участвует в общем confidence профиля;
- `expression` — состояние конкретного кадра, например раскрытие рта;
- `diagnostic` — контроль ракурса, выравнивания или покрытия сетки.

Expression и diagnostic сохраняются в отчете, но не определяют уверенность
морфологического профиля.

## Правило расширения

Новый признак добавляется только вместе с:

1. стабильным именем;
2. явной формулой;
3. зоной и единицами;
4. минимальными требованиями к данным;
5. тестом инвариантности к переносу и масштабу;
6. проверкой повторяемости по серии фотографий.
